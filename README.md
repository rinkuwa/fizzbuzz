# 第３回
## タイトル　fizzbuzz
**機能要件**
* 整数の出力:指定された範囲の整数を順に出力する。
* 3の倍数の処理:数字が3の倍数のときは、「Fizz」を出力する。
* 5の倍数の処理:数字が5の倍数のときは、「Buzz」を出力する。
* 3と5の倍数の処理:数字が3の倍数かつ5の倍数のときは、「FizzBuzz」を出力する。
* その他の数の処理:3の倍数でも5の倍数でもない場合は、その数字をそのまま出力する。
